## 📖 Project Overview

This project is a CRUD (Create, Read, Update, Delete) Blog Management System that allows users to manage blog posts through a clean and responsive interface.

The frontend is built using HTML, CSS, and JavaScript, while the backend is developed using Node.js and Express.js.

---

## ✨ Features

- ➕ Add Blog
- 📄 View Blogs
- ✏️ Edit Blog
- 🗑️ Delete Blog
- 🔗 RESTful APIs
- 🌐 Fetch API Integration
- 📱 Responsive Design
- 🎨 Smooth Animations & Hover Effects

## 🚀 How to Run the Project

### Clone the Repository

```bash
git clone https://github.com/swagath088/codomax-fullstack-internship.git
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
node index.js
```

Open your browser and visit:

```
http://localhost:3000
```
